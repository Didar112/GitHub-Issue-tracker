
# 1. Difference between var, let and const is: 
1) We can never change the value of a variable that is declared by const, where, it is possible to change the value of those that are declared with let and var.
2) let and const maintains a TDZ, means they cannot be accessed before their declaration, but var has no TDZ


# 2. spread operator
spread operator is used when we want to access all the individual elements of an array or object. It is often used to create a copy of such array and objects without tempering the original one.

# 3. What is the difference between map(), filter(), and forEach()
a) map transforms each element of an iterable object\
b) filter selects elements based on some conditions and returns a new array made with those selected elements\
c) forEach is used to perform a certain function for all individual elements of an iterable object.
